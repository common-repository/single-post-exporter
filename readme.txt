Export Posts, Pages and Custom Posts using single click.

== Description ==

#### you can simply export any single or multiple posts, pages and custom posts by just one click. You can import exported posts on other wordpress installations. 


== Installation ==

1. Upload the `single-post-exporter` folder to the directory `/wp-content/plugins/`.
2. Activate the plugin using the 'Plugins' menu in WordPress.

## How to use

1. First Activate Plugin.
2. Just click on export link in posts to export specific post, page and custom post.
